
**Calculator Project**

- **Description:** A simple, client-side calculator web project with a clean UI and basic arithmetic functionality. It's implemented with HTML, CSS, and JavaScript and intended as a small learning/demo app.

**Structure**
- **`index.html`**: Main entry page for the calculator UI.
- **`assests/css/`**: Stylesheets (`style.css`, `utility.css`) for layout and appearance.
- **`assests/js/script.js`**: JavaScript logic for the calculator.
- **`assests/videos/`**: Optional media assets used by the project.

**Features**
- **Basic arithmetic:** add, subtract, multiply, divide.
- **Responsive UI:** simple layout that works on desktop and mobile browsers.
- **Standalone:** No build tools or server required â€” runs in any modern browser.

**How to Use**
- **Open:** Double-click `index.html` or run `start index.html` from PowerShell in the project directory to open the app in your default browser.
- **Interact:** Click the on-screen buttons or use your keyboard to enter numbers and operations.

**Development**
- **Edit files:** Update UI in `index.html`, styles in `assests/css/`, and behavior in `assests/js/script.js`.
- **Preview:** Open `index.html` after changes to see them immediately. No build step required.
